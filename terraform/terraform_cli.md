# Terraform commands

## Terraform main commands
```
terraform -help
```

```
terraform fmt
```

```
terraform init
```

```
terraform validate
```

```
terraform plan
terraform plan -out <file_name>
terraform plan -destroy
```

```
terraform apply
```

```
terraform destroy
```

```
terraform providers
```

```
terraform state list
```

```
terraform state show <resource>
```

```
terraform show
```
