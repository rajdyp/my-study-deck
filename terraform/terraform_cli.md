# Terraform commands

## Terraform main commands
```yaml
terraform -help
```

```yaml
terraform init
```

```yaml
terraform fmt
```

```yaml
terraform console
```

```yaml
terraform validate
```

## Plan
```yaml
terraform plan
```
```yaml
terraform plan -out <file_name>
```
```yaml
terraform plan -destroy
```

## Apply
```yaml
terraform apply
```
```yaml
terraform apply -refresh-only
```
```yaml
terraform apply -auto-approve
```

## Destroy
```yaml
terraform destroy
```

```yaml
terraform providers
```

```yaml
terraform state list
```

```yaml
terraform state show <resource>
```

```yaml
terraform show
```

```yaml
terraform apply -replace <resource>
```
