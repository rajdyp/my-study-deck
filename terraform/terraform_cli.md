# Terraform commands

## Terraform main commands
```yaml
terraform -help
```

```yaml
terraform fmt
```

```yaml
terraform console
```

```yaml
terraform init
```

```yaml
terraform validate
```

```yaml
terraform plan
terraform plan -out <file_name>
terraform plan -destroy
```

```yaml
terraform apply
terraform apply -refresh-only
```

```yaml
terraform destroy
```

```yaml
terraform providers
```

```yaml
terraform state list
```

```yaml
terraform state show <resource>
```

```yaml
terraform show
```

```yaml
terraform apply -replace <resource>
```
